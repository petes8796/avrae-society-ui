# Avrae UI Clone

This is a ready-to-deploy React frontend (Create React App) with Tailwind + Framer Motion for the Avrae Society UI clone. Paste into your Vercel project or import as a repo.

## To deploy (recommended)
1. Upload this folder to GitHub and connect to Vercel.
2. In Vercel, set root to project (no special root directory) and run build.

## Notes
- After first deploy, add environment variables if any backend URL is required.
